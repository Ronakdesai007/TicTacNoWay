# Tic Tac No Way

Tic Tac No Way is a simple and engaging Tic Tac Toe game that offers two modes: single-player mode against the computer and two-player mode where you can challenge your friends.

## Features

- **Single Player Mode**: Play against an AI opponent.
- **Two Player Mode**: Challenge a friend to a classic game of Tic Tac Toe.

## Technologies Used

- **HTML**: For the structure of the game.
- **CSS**: For styling the game interface (designed with Stylus).
- **JavaScript**: For implementing the game logic.

## Installation Instructions

To play the game, simply download the project files and run the `index.html` file in your web browser. No additional setup is required.

## Usage

1. Open the `index.html` file in your web browser.
2. Choose your game mode (Single Player or Two Player).
3. Follow the on-screen instructions to play the game.
